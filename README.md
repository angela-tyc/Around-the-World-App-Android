# Around the World 🌍

A tourist app for browsing and tracking travel destinations.  
**CS 329E Mobile Computing — Spring 2022, UT Austin**  
**Team Double HCs:** Jiaxin Huang, Grace Huang, Angela Chen, Micah Chow

## Overview
Around the World lets users explore tourist destinations around the world,
save favorites, and record their own travel itineraries.

## Features
- **Home** — Browse destinations in grid or list view
- **Search** — Find sites by name with live suggestions
- **Site Details** — View images and descriptions of each destination
- **Map** — See destination locations via Google Maps (Normal, Hybrid, Satellite, Terrain views)
- **Favorites** — Save sites you like with a heart icon
- **Itinerary** — Log places you've visited or want to visit, with dates and notes

## Requirements Met
- Persistent data (itinerary saved between sessions)
- Animations (splash screen, button scaling, fragment fade transitions)
- GPS / Google Maps integration

## Built With
- Android Studio
- Java
- Google Maps SDK

## How to Run
1. Clone the repo
2. Open in Android Studio
3. Add your Google Maps API key to `local.properties` or `AndroidManifest.xml`
4. Run on an emulator or physical Android device
